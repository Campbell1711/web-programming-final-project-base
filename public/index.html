Corpus

Most documents are derived from the works of William Shakespeare, made available by Open Literature and preprocessed into JSON format through the CS446 course, taught by David Fisher.

Search Types

Content-based search is done by returning documents that contain all of the query tokens, splitting the text into tokens using whitespace.

Scene-based search will return all documents whose scene title is equal to the query. Most of these will be of the form play:int.int, but this is not a requirement.

Play-based search will return all documents whose play title is equal to the query.

Tag-based search will return all documents who possess the tag given in the query.

All search types are case insensitive.
